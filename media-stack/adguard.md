# AdGuard Home

AdGuard Home can provide DNS filtering and visibility into blocked domains for a homelab.

## Defensive Value

- Reduces exposure to known unwanted domains.
- Provides DNS query visibility for troubleshooting.
- Helps practice safe log review and privacy-aware monitoring.

## Security Considerations

- Restrict the admin interface to trusted local access.
- Use strong authentication.
- Avoid publishing real client names, query logs, or internal addresses.
- Review upstream DNS choices and privacy implications.
